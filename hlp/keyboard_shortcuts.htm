Keyboard Shortcuts

Ctrl+O Select Originals
Ctrl+P Processing Options
Ctrl+L Rotate Left
Ctrl+R Rotate Right
Ctrl+A Adjust Rotation
F1 Help
F3 Previous Picture
F4 Next Picture
F8 Crop/Save Picture   (You can also double click inside picture area)
F11 True Size
Shift+F11 Size to Fit
Shift+F8 Resize All
Double Click Save Picture - You must double click inside the picture area. If you want to crop and save the picture, you must double click inside the cropping rectangle.
Alt+F4 Exit the program
