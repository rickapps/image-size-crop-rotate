Toolbar Buttons

Select Originals - Choose the folder where  your images reside. The program will display all jpeg files found in that folder. You can also drag and drop a folder or image on the program. If you drop an image on the program, all images in its parent folder will be displayed.  
Processing Options - (Advanced Users) Select the folder to store your reduced, processed images. The drop down list gives you three choices of where to store your processed photos:
  1. In a folder on your desktop called 'FitToList Photos'
  2. In a folder called 'Processed', located inside the folder containing your original photos 
  3. In a folder you pick by clicking the 'Browse' button. The drop down will display the last folder you picked. If processed photos cannot be stored in the chosen folder, the program will put them in the 'FitToList Photos' folder on your desktop instead.

You can also set the maximum dimensions for processed images. Original pictures that exceed these dimensions will be reduced to make them suitable for web or email use. The reduced images will be stored in the processed folder you select. The original images will not be affected. If an original is already smaller than the size you select, it will be copied to the processed folder without a change in size.

Next Picture - Moves to the next picture in the selection list on the left side of the screen. If the current picture is an original (identified by a check mark), the selection is changed to the next original. If the current image is processed (identified by a circle), the selection is changed to the next processed image. If there are no more processed images, the selection is changed to the next original image. 
Previous Picture - Works the same as 'Next Picture', but in reverse.
True Size - Displays the image without any resizing. The picture is displayed using the dimensions produced by your camera or scanner.
Size to Fit - Pictures are resized to fit the current screen size. The image is sized to make the entire image visible without scrolling. Only the displayed image is reduced. The image stored on your computer is not affected. For large pictures, program performance might be reduced. If the program response time is too slow, try switching to 'True Size'.
Crop/Save Picture -  Drag your mouse anywhere within an image area to create a cropping rectangle. Click the toolbar button to save the cropped area to the processed folder. You can also double click inside the cropping area to save or you can press the F8 function key.  
Rotate Right - Rotate the image clockwise ninety degrees. The rotated image is saved to your computer. The original image is overwritten.
Rotate Left - Rotate the image counter-clockwise ninety degrees. The rotated image is saved to your computer. The original image is overwritten.
Adjust Rotation - Slightly adjust the angle of rotation for an image. Use this function to level an image before you crop it. The original image is not modified.
Flip Vertical - Reverse the image up and down. The original image is not modified.
Flip Horizontal - Reverse the image left and right. This creates a mirror image of the original. The original image is not modified.
Resize All - Resize all images that are larger than the maximum set size. Your original photos will not be affected. Reduced photos will be stored in the Processed folder that you select. You can change both the storage location of processed photos and the maximum size settings from this screen. 
View Processed - View the contents of the current 'Processed' folder. You can also click the downward pointing triangle to the right of the toolbar button to view the contents of the current 'Originals' folder.
Help - Display this help document.

Keyboard Shortcuts

Ctrl+O Select Originals
Ctrl+P Processing Options
Ctrl+L Rotate Left
Ctrl+R Rotate Right
Ctrl+A Adjust Rotation
F1 Help
F3 Previous Picture
F4 Next Picture
F8 Crop/Save Picture   (You can also double click inside picture area)
F11 True Size
Shift+F11 Size to Fit
Shift+F8 Resize All
Double Click Save Picture - To both crop and save the picture, double click inside the cropping rectangle. If you want to save the entire picture without cropping, double click anywhere on the picture that is outside the cropping rectangle.
Alt+F4 Exit the program

Mouse Shortcuts

Right click your mouse on any image name in the selection tree on the left side of the screen to display a pop-up menu. Pop-up menu choices are; 'Select Originals', 'Processing Options', and 'Refresh'. Refresh will read image names from the 'Originals' folder and return the selection list to its initial state.

Right click your mouse on the currently displayed image on the right side of the screen to display a pop-up menu. Menu choices are; 'Restore', 'Crop/Save Picture', 'Next Picture', 'Previous Picture', 'Rotate Left', and 'Rotate Right'. 'Restore' will return the image to its last saved state.

Drag an drop a folder containing images or a jpeg image onto the program to quickly select an Originals folder.
